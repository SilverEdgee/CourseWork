Руководство пользователя Hand Gesture IDE Controller
============================================

⚠️ ВАЖНО: Полная функциональность доступна только на Linux!

1. Установка и запуск
--------------------

1.1 Установка на Linux (полная функциональность)
- Убедитесь, что установлен Python 3.8 или выше
- Создайте виртуальное окружение Python
- Установите все зависимости из requirements.txt
- Установите X Server: sudo apt-get install xserver-xorg
- Разрешите доступ к X Server: xhost +
- Подключите веб-камеру к компьютеру

1.2 Установка на Windows/MacOS (ограниченная функциональность)
- Установите Python 3.8 или выше
- Создайте виртуальное окружение
- Установите базовые зависимости
- Подключите веб-камеру
- Примечание: распознавание жестов может не работать

1.3 Запуск приложения
- Запустите qt_app.py
- В главном окне нажмите "ЗАПУСТИТЬ КАМЕРУ"
- Выберите камеру из выпадающего списка, если доступно несколько

2. Основной интерфейс
--------------------

2.1 Панель камеры и управления
- Выбор камеры
- Кнопка запуска/остановки камеры
- Кнопка справки по жестам

2.2 Настройки распознавания (Linux)
- Выбор чувствительности (низкая/средняя/высокая)
- Настройка задержки между действиями
- Кнопка применения настроек

2.3 Панель отображения
- Окно с видео от камеры
- Визуализация распознанных жестов (Linux)
- Текущий жест и действие
- Лог событий

3. Запись новых жестов (только Linux)
---------------------

3.1 Подготовка к записи
- Переключитесь в "Режим записи жестов"
- Выберите номер жеста (0-9)
- Убедитесь, что рука хорошо видна в кадре

3.2 Процесс записи
- Покажите жест перед камерой
- Нажмите кнопку "ЗАПИСАТЬ ЖЕСТ"
- Повторите несколько раз с разными положениями руки
- Следите за счетчиком записанных кадров

3.3 Завершение записи
- Переключитесь обратно в нормальный режим
- Или нажмите ESC для быстрого выхода из режима записи

4. Настройка действий (Linux)
-------------------

4.1 Привязка действий к жестам
- Выберите жест из выпадающего списка
- Выберите желаемое действие
- Нажмите "СОХРАНИТЬ ДЕЙСТВИЕ"

4.2 Доступные действия
- Комбинации клавиш
- Клики мыши (левый, правый, двойной)
- Прокрутка
- Запуск приложений
- Системные команды

5. Настройка чувствительности (Linux)
---------------------------

5.1 Уровни чувствительности
- Низкая: быстрое распознавание, возможны ложные срабатывания
- Средняя: сбалансированный режим (рекомендуется)
- Высокая: точное распознавание, меньше ошибок

5.2 Когда менять чувствительность
- Низкая: при хорошем освещении и четких жестах
- Средняя: для большинства ситуаций
- Высокая: при плохом освещении или сложных жестах

6. Советы по использованию
------------------------

6.1 Оптимальные условия (Linux)
- Хорошее освещение
- Контрастный фон
- Расстояние до камеры 0.5-1.5 метра
- Рука полностью в кадре

6.2 Частые проблемы

Linux:
- Камера не запускается
  * Проверьте подключение
  * Закройте другие приложения с камерой
  * Проверьте права доступа

- Жесты не распознаются
  * Улучшите освещение
  * Настройте чувствительность
  * Держите руку в поле зрения камеры

- Действия не выполняются
  * Проверьте настройки действий
  * Увеличьте задержку между действиями
  * Проверьте выполнение команды xhost +

Windows/MacOS:
- Ограниченная функциональность
- Распознавание жестов может не работать
- Используйте Linux для полной функциональности

6.3 Рекомендации по записи жестов (Linux)
- Записывайте не менее 30 кадров для каждого жеста
- Меняйте положение и ориентацию руки
- Используйте разное освещение
- Проверяйте качество распознавания после записи

7. Горячие клавиши
-----------------

- ESC: выход из режима записи
- Другие горячие клавиши настраиваются в разделе действий

8. Дополнительные возможности
--------------------------

8.1 Справка по жестам
- Доступ через кнопку "СПРАВКА ПО ЖЕСТАМ"
- Показывает все доступные жесты и их действия
- Обновляется при изменении настроек

8.2 Лог событий
- Отображает историю действий
- Помогает в отладке проблем
- Показывает статус операций

9. Безопасность
-------------

9.1 Права доступа (Linux)
- Приложению требуются права на:
  * Доступ к камере
  * Управление клавиатурой и мышью (X Server)
  * Запуск других приложений

9.2 Конфиденциальность
- Видео не сохраняется
- Данные о жестах хранятся локально
- Нет передачи данных в интернет
