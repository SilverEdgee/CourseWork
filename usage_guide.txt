===============================
РУКОВОДСТВО ПО ИСПОЛЬЗОВАНИЮ HAND GESTURE CONTROLLER
===============================

ОБЩЕЕ ОПИСАНИЕ
-------------
Hand Gesture Controller - это приложение для распознавания жестов рук с помощью камеры и выполнения различных действий в IDE на основе распознанных жестов. Приложение использует библиотеку MediaPipe для отслеживания рук и классификатор жестов на основе TensorFlow.

НАЧАЛО РАБОТЫ
------------
1. Запустите приложение
2. Выберите камеру из выпадающего списка (если у вас несколько камер)
3. Нажмите кнопку "Запустить камеру"
4. После успешного запуска камеры вы увидите изображение с камеры в левой части экрана

ИНТЕРФЕЙС ПРИЛОЖЕНИЯ
------------------

ЛЕВАЯ ПАНЕЛЬ (НАСТРОЙКИ)
------------------------

Раздел "Настройки камеры":
------------------------
- "Камера" - выпадающий список для выбора камеры
- "Запустить камеру" / "Остановить камеру" - кнопка для включения/выключения камеры
- "Показать справку по жестам IDE" - кнопка для отображения информации о доступных жестах и их функциях в IDE

Настройки MediaPipe:
- "Статичный режим" - флажок для включения статического режима (рекомендуется отключить для более плавного отслеживания)
- "Порог обнаружения" - ползунок для настройки минимальной уверенности обнаружения руки (выше = точнее, но может пропускать руки)
- "Порог трекинга" - ползунок для настройки минимальной уверенности трекинга (выше = стабильнее, но может быть менее гибким)
- "Задержка между действиями" - время в секундах между выполнением действий одного типа (предотвращает случайные повторные срабатывания)
- "Применить настройки" - кнопка для применения изменений настроек MediaPipe

Раздел "Распознавание жестов":
----------------------------
- "Режим" - выпадающий список для выбора режима работы:
  * "Нормальный режим" - обычный режим распознавания жестов
  * "Запись данных для жестов" - режим для сбора данных для обучения новых жестов
  * "Запись данных для движений" - режим для сбора данных для обучения распознавания движений

- "Доступные жесты" - список всех доступных в системе жестов
- "Добавить" - кнопка для добавления нового жеста (требует дополнительного обучения модели)
- "Переименовать" - кнопка для переименования выбранного жеста
- "Удалить" - кнопка для удаления выбранного жеста

Раздел "Действия для жестов":
--------------------------
- "Жест" - выпадающий список для выбора жеста, которому нужно назначить действие
- "Действие" - выпадающий список для выбора действия, которое будет выполняться при распознавании жеста
- "Сохранить настройку действия" - кнопка для сохранения связи между жестом и действием

ПРАВАЯ ПАНЕЛЬ (ОТОБРАЖЕНИЕ И ИНФОРМАЦИЯ)
--------------------------------------
- "Оригинальное изображение" - видеопоток с камеры без обработки
- "Обработанное изображение" - видеопоток с отображением распознавания жестов

Раздел "Информация о распознавании":
---------------------------------
- "Текущий жест" - показывает текущий распознанный жест
- "Текущее действие" - показывает действие, соответствующее текущему жесту
- "Лог событий" - список последних событий и действий в приложении

Кнопки внизу:
- "Запустить распознавание" - кнопка для запуска/остановки распознавания жестов
- "Сохранить конфигурацию" - кнопка для сохранения текущих настроек
- "Загрузить конфигурацию" - кнопка для загрузки сохраненных настроек

НАСТРОЙКА ТОЧНОСТИ РАСПОЗНАВАНИЯ ЖЕСТОВ
------------------------------------
Для повышения точности распознавания жестов:

1. Настройте параметры MediaPipe:
   - Увеличьте "Порог обнаружения" (рекомендуется 0.7-0.8)
   - Увеличьте "Порог трекинга" (рекомендуется 0.5-0.7)
   - Нажмите "Применить настройки"

2. Обеспечьте хорошее освещение – равномерное освещение руки улучшает качество распознавания

3. Держите руку на оптимальном расстоянии от камеры (30-60 см)

4. При необходимости обучите модель распознавания своими данными:
   - Выберите режим "Запись данных для жестов"
   - Соберите примеры жестов для каждого класса
   - Обучите модель с помощью ноутбука keypoint_classification.ipynb

5. Увеличьте "Задержку между действиями" (1-2 секунды), если действия выполняются слишком часто или случайно

ДОСТУПНЫЕ ЖЕСТЫ И ИХ ФУНКЦИИ В IDE
--------------------------------
- "Open" (Открытая ладонь) - Tab - Подтверждение автодополнения кода
- "Close" (Закрытая ладонь) - Ctrl+X - Вырезать выделенный текст
- "Pointer" (Указательный палец) - Home - Перемещение в начало строки
- "OK" (Палец-кольцо) - Shift+End - Выделение от курсора до конца строки
- "Thumb Up" (Большой палец вверх) - Ctrl+S - Сохранить файл
- "Peace Sign" (Жест мира/V) - Ctrl+C - Копирование выделенного текста
- "Thumb Down" (Большой палец вниз) - Ctrl+Z - Отменить последнее действие
- "Rock" - F5 - Запуск отладки
- "Pistol" - Ctrl+/ - Закомментировать/раскомментировать строку

ГОРЯЧИЕ КЛАВИШИ
-------------
- 0-9: Установка номера во время записи данных
- N: Переключение в нормальный режим
- K: Переключение в режим записи жестов
- H: Переключение в режим записи истории движений

ПРИМЕЧАНИЯ
---------
- Для обучения новых жестов требуется запустить обучение модели через ноутбуки Jupyter
- Файлы конфигурации жестов хранятся в gesture_actions_config.json
- Для редактирования меток жестов можно изменить файл model/keypoint_classifier/keypoint_classifier_label.csv
